    API Guide

    Compute Studio offers a REST API for creating simulations, viewing existing simulations, and viewing the input parameters for a given model. This is intended for users who would like to build applications on top of the Compute Studio interface or create simulations programmatically. An API token is required for requests that modify data. Find out how to get your token here.

    More information about the data formats that are shown below can be found in the Compute Studio publishing documentation on inputs and outputs.

    This guide details the Compute Studio API endpoints and schema. A more practical Python example is also provided.

    /[owner]/[title]/api/v1/

    Used for creating simulations.

    Supports POST HTTP actions.

    Create simulation

    POST hdoupe/Matchups/api/v1/

    Example:

    {
  "meta_parameters": {
    "use_full_data": true
  },
  "adjustment": {
    "matchup": {
      "pitcher": "Max Scherzer"
    }
  }
}

    Response:

    HTTP/1.1 201 Created
Allow: GET, POST, HEAD, OPTIONS

    Used for getting simulations.
